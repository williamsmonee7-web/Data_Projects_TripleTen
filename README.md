# Data_Projects_TripleTen
Projects I worked on while attending the Data Analysis TripleTen Bootcamp.


| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | Fresh Beats Business Insights Report| This report analyzes artist promotion effectiveness, user engagement and conversion trends, and genre promotion opportunities at Fresh Beats to provide stakeholders with actionable insights and recommendations for increasing artist visibility, improving user retention, and driving revenue growth. |
| 2 | Manhattan Airbnb: What to Buy, Where to Buy, and Why | Analyzed Manhattan Airbnb rental data using data cleaning techniques, pivot tables, and revenue calculations to identify the most attractive neighborhoods and property sizes for vacation rentals, estimate annual earnings, and provide data-driven investment recommendations. |
| 3 | Tracking User Journeys: Funnel & Cohort Insights | Analyzed e-commerce user activity logs by building a conversion funnel, preparing purchase data for cohort analysis, calculating customer retention rates, and presenting insights on user behavior, conversion performance, and customer retention trends through organized spreadsheet reporting. |
| 4 | Decoding Shopify App Success: Ratings, Reviews, and Developer Impact | Used Tableau to analyze Superstore sales, profits, advertising opportunities, and product return rates, creating data visualizations that identified key profit drivers, loss-making products, high-value advertising targets, and operational improvements to increase overall profitability. |
| 5 | Behind Shopify App Success: A Marketplace Deep Dive| Used Power BI to analyze the Shopify App Store ecosystem by exploring app trends, review patterns, developer responsiveness, and app performance metrics, creating interactive visualizations that identified key factors associated with app success and user satisfaction. |
